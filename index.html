React

What is React?

  • A open-source component-based approach JavaScript library for building user interfaces
  • Forces you to think in terms of components.
  • Created by Facebook in 2013

Why do Developers use React?

  • Easy to learn and use
  • Less code -> more functionality
  • Reusable components
  • Performance Enhancement by Virtual DOM
  • Flexible
  • Easy to test
